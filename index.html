ootstrap Element

Ett element inom html är en typ av komponent som används inom webbsidor.

HTML

HTML är ett scriptspråk som används för att skapa och ange den grundläggande strukturen för en hemsida. HTML är tillsammans med HTTP och IP standarden för www-hemsidor.

ID

ID är ett namn som man ger till ETT specifikt element. Ett ID kan även förändra utseendet av ett element.

Class

Class fungerar som ett ID som man kan ge till flera element (inte bara ett) och precis som ett ID kan en klass ändra utseendet av ett element.

Meta

En Meta(tagg) är ett element som används för att ange vilken typ av data som används inom en HTML-fil. Exempelvis vilket charset som används.

Responsive

Att en hemsida är responsiv innebär att den kan ändra sin layout utifrån vilket skärmstorlek eller upplösning som användaren har. Det innebär även att den ska kunna anpassa sig till både mobil- och datorenheter

HTTP

HTTP (Hypertext transfer protocol) är ett protokoll som används för att överföra webbsidor (html-, css- och javascript-filer) från en server till en klient.

HTTPS

HTTPS (Hypertext transfer protocol secure) är ett liknande protokoll som HTTP men som överför krypterad data i form av TSL-encryption. Detta innebär att andra personer inte kan ta reda på vad som skickats mellan din dator och webbservern utan att ha nyckeln för att kunna dekryptera.

DNS

DNS är ett system som översätter domännamn till ip-adresser.

CDN

Ett CDN (content delivery network) är en grupp geografiskt utspridda nätverk som används för att minska latency när man skickar paket (information).

Compressed data

Datakompression är ett sätt att göra om data och minska information utan att senare förlora data när man extrahera. Data görs om så att enbart vissa delar återstår som senare kan användas flera gånger för att återuppbygga den ursprungliga datan.

Local bootstrap

Lokal bootstrap innebär att man har en egen version av bootstrap nedladdad istället för att länka över nätet.

JQuery

JQuery är ett javascript bibliotek som används för att förenkla olika processer inom utvecklingen. Ett exempel på detta är AJAX-operationer som används för att man kan ska kunna ändra kod i en hemsida i real-tid

Flexbox

Flexbox är en modul inom CSS som används för att skapa 1D layouter (antingen rader eller kolumner) med items som anpassar sig utifrån skärmstorlek och upplösning. Det är därför enklare att exempelvis centrera saker på hemsidan.

Grid

Grid är en modul inom CSS som likt flexbox som låter dig skapa rutnätslayouter som exempelvis kan visa information. Skillnaden mellan grid och flexbox är att grid även kan användas för att skapa 2D layouter, alltså rader och kolumner samtidigt.

Code snippets

Code snippets är delar av kod/program som används för att man enklare ska kunna köra om en viss del av kod utan att behöva köra hela programmet.

WASM

WASM (WebAssembly) är ett binärt kodformat som körs i moderna webbläsare. Det används för att tillåta lågnivå-språk som exempelvis c/c++ att kunna köras på webben.

Chromium

Chromium är en webbläsare med öppen källkod från Google.

SVG

SVG är ett vektorgrafiskt format för 2d bilder. SVG filer defineras genom xml-textfiler.

PNG24

PNG24 är ett png-format som stödjer rgb-färger. Varje kanal går från 0-255 (8 bit) och 3 färgkanaler resulterar då i 24 bit (3x8), därav PNG24.
